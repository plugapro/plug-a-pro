# ServiceMen App Strategy Docs

This folder contains the current strategy working set for `ServiceMen App`.

## Documents

1. [Persona Clarification Document](./persona-clarification-document.md)
2. [MVP Scope Prioritisation Document](./mvp-scope-prioritisation-document.md)
3. [Role-by-Role Feature Requirements Matrix](./role-by-role-feature-requirements-matrix.md)
4. [WhatsApp Journey Specification](./whatsapp-journey-specification.md)

## Purpose

These documents are intended to support:

- founder-level product and operating model decisions
- UX and service design work
- role-based workflow design
- admin and field operations planning
- MVP scope control
- WhatsApp and PWA journey definition

## Suggested Reading Order

1. `persona-clarification-document.md`
2. `mvp-scope-prioritisation-document.md`
3. `role-by-role-feature-requirements-matrix.md`
4. `whatsapp-journey-specification.md`

## Notes

- The documents assume a mobile-first, WhatsApp-enabled operating model.
- The intended market context is South Africa.
- The content is written for product, operations, design, and commercial planning rather than implementation detail.
