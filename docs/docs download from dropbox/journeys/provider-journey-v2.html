Plug A Pro — Service Provider Journey (v2)

Updated operating model: WhatsApp-first · WhatsApp-complete · PWA-optional. Provider can complete the full lifecycle inside WhatsApp. The PWA is for richer dashboards, profile management, credit history, image gallery, and document management — never required for transactional flow.
