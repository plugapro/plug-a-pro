Plug A Pro — Customer Journey (v2)

Updated operating model: WhatsApp-first with structured PWA handoff. Customer starts in WhatsApp, gets handed to a signed PWA link for richer capture (photos · structured address · urgency / preferences) and for the shortlist comparison & provider selection screen — the part WhatsApp can't render. Tracking and notifications flow through both channels off the same backend state.

WhatsApp action PWA action Backend / system Admin / Ops Credit event Privacy boundary Decision Exception / error
