# Unrequested OTP Fraud Response Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship the unrequested OTP fraud-response layer for Plug A Pro without replacing Supabase Auth as the OTP authority.

**Architecture:** Add a shadow security layer around the existing Supabase OTP send and session-cookie paths. Store challenge/security state in additive Prisma tables with deny-by-default RLS, gate app session issuance through one shared helper, and expose report/step-up/admin flows behind feature flags.

**Tech Stack:** Next.js App Router, TypeScript, Prisma, Supabase Auth, WhatsApp Cloud API webhook/template delivery, Upstash rate limiting, Vitest, Playwright smoke coverage.

Implementation details:
- Use `createHmac('sha256', pepper)` for OTP/report signatures.
- Use `timingSafeEqual` when comparing HMACs.
- Use `hkdfSync('sha256', pepper, 'pap-step-up-token:v1', 'plug-a-pro/security-step-up', 32)` when `STEP_UP_COOKIE_KEY` is unset.
- Decode `STEP_UP_COOKIE_KEY` as base64url first, then base64; require 32 bytes.
- Use AES-256-GCM with a fresh 12-byte IV.
- Cookie header is `pap-step-up-token=[encrypted-token]; HttpOnly; SameSite=Lax; Path=/; Max-Age=600` plus `; Secure` in production.

- [ ] **Step 5: Implement trusted IP extraction**

Create `field-service/lib/request-ip.ts`:

```ts
import { isIP } from 'node:net'
import type { NextRequest } from 'next/server'

const PRIVATE_RANGES = [
  /^10\./,
  /^127\./,
  /^169\.254\./,
  /^172\.(1[6-9]|2\d|3[0-1])\./,
  /^192\.168\./,
  /^::1$/,
  /^fc/i,
  /^fd/i,
  /^fe80:/i,
]

function isPublicIp(value: string): boolean {
  const ip = value.trim()
  if (!ip || PRIVATE_RANGES.some((range) => range.test(ip))) return false
  return isIP(ip) !== 0
}

export function trustedClientIpFromHeaders(headers: Pick<Headers, 'get'>): string | null {
  const forwarded = headers.get('x-forwarded-for')
  if (forwarded) {
    for (const part of forwarded.split(',')) {
      const ip = part.trim()
      if (isPublicIp(ip)) return ip
    }
  }

  const realIp = headers.get('x-real-ip')?.trim()
  return realIp && isPublicIp(realIp) ? realIp : null
}

export function trustedClientIp(request: NextRequest | Request): string | null {
  return trustedClientIpFromHeaders(request.headers)
}
```

This helper intentionally reads only trusted proxy headers and never accepts an IP field from JSON bodies.

```ts
export async function POST() {
  const pending = cookies().get('pap-step-up-token')?.value
  const invalid = NextResponse.json({ ok: false, restartSignIn: true }, { status: 401 })
  invalid.headers.set('Set-Cookie', clearPendingStepUpCookieHeader())

  if (!pending) return invalid
  const decrypted = decryptPendingStepUpCookie(pending)
  if (!decrypted.ok) return invalid

  await completeStepUp(decrypted.payload.phoneE164, decrypted.payload.userId)
  const response = NextResponse.json({ ok: true })
  response.headers.append('Set-Cookie', buildSessionCookieHeader(decrypted.payload.accessToken, decrypted.payload.maxAge))
  response.headers.append('Set-Cookie', clearPendingStepUpCookieHeader())
  return response
}
```

```ts
import { NextResponse } from 'next/server'
import { pruneTerminalOtpChallenges } from '@/lib/otp-security'

export async function GET(request: Request) {
  const authHeader = request.headers.get('authorization')
  if (!process.env.CRON_SECRET || authHeader !== `Bearer ${process.env.CRON_SECRET}`) {
    return new NextResponse('Unauthorized', { status: 401 })
  }

  const startedAt = Date.now()
  const result = await pruneTerminalOtpChallenges()
  const durationMs = Date.now() - startedAt

  console.info(JSON.stringify({
    event: 'otp.challenge.pruned',
    deleted: result.deleted,
    durationMs,
  }))

  return NextResponse.json({ ok: true, deleted: result.deleted, durationMs })
}
```
